

实现一个本地RAG系统，基于自己投喂的语料库，实现私人rag问答助手。

<!-- 基础架构 -->
1. 技术栈：
前端：react + TypeScript + tailwind + Vite + Framer Motion + axios
后端：python（FastAPI）
模型底座：Gemma-4-E4B-GGUF
向量数据库：ChromaDB
推理核心：llama-cpp-python
补充一个 “上下文压缩（Context Compression）” 模块。因为 4060 的显存只有 8GB，如果你喂的文档太长，显存会爆掉。

2. 整体主题风格默认选择黑灰色系列，参考海外主流炫酷的站点风格。主题色用一个provider配置，支持暗系和亮系的切换

3. 新增.gitignore文件，配置基本配置

4. 图标库使用fontawesome


<!-- 项目具体需求 -->
5. 整体页面布局上小俩部分
下方固定高度，用于用户输入框
上方占据页面除底部外的所有高度，主要是与大模型的交互

6.项目结构：前端代码放进frontend，后端扔进backend
